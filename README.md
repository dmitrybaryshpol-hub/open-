# Telegram Preview Bot

Цей проект створює простого Telegram-бота, який приймає посилання на пости в Instagram, Twitter/X або TikTok і повертає попередній перегляд.

## Як запустити

1. Встановіть Python 3.9+.
2. Встановіть залежності:

```bash
pip install -r requirements.txt
```

3. Створіть бот у BotFather і отримайте токен.
4. Збережіть токен у змінній середовища:

```bash
set TELEGRAM_TOKEN=ваш_токен
```

5. Запустіть бота:

```bash
python bot.py
```

## Використання

Надішліть боту посилання на пост у Instagram, Twitter/X або TikTok, і він відповість превью із заголовком та зображенням (якщо воно доступне).
